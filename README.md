# mllm-3

Introducing the next generation of MLLM models by the mincoffical Team.
With a new expanded corpus for more general things.

P= Pages of data

Current Sizes: 

MLLM-3 (108P)

MLLM-3-Flash (50P)

Planned Sizes: 
MLLM-3 Max (700P)
MLLM-3 Micro (10P)
MLLM-3-Nano (3P)


This model is less "focused" but more random than MLLM-2 and 2.5, while maintaining quality. Because of this, it works great as an N-gram based autocomplete system.
